# nonebot-plugin-recall

群聊防撤回插件（OneBot v11）。

当白名单群内有人撤回**自己的**消息时，机器人会发送：

- 文本/图片等消息：`用户名撤回了一条消息：...`（图片会以真实图片段发送）
- 语音消息：先发 `用户名撤回了一条语音`，再单独发送原语音



## 配置

在 `.env` 中添加：

```env
RECALL_GROUP_WHITELIST=[123456789,987654321]
```

- 只有在白名单中的群才会触发防撤回。
- 默认白名单为空（即不启用任何群）。

## 说明

- 插件优先使用内存缓存恢复消息内容，缓存中没有时会尝试调用 `get_msg`。
- 仅处理最近 2 分钟内发送后被撤回的消息，超时消息将忽略。
- 如果原消息无法获取，会发送：`用户名撤回了一条消息：[内容未捕获]`。
- 超级表情目前无法正常发送
